## 1. Executive Summary

### What is Vesta Lumina?

Vesta Lumina is an **enterprise-grade vacation rental management system** designed specifically for the Croatian and European hospitality market. The system digitizes and automates the entire guest experience lifecycleâ€”from booking to checkoutâ€”while providing property owners with powerful management tools.

### Key Value Propositions

| Stakeholder | Value Delivered |
|-------------|-----------------|
| **Property Owners** | Centralized management, automated workflows, legal compliance |
| **Guests** | Self-service check-in, instant information access, AI concierge |
| **Cleaning Staff** | Mobile task management, digital checklists, photo documentation |
| **Super Admins** | Multi-property oversight, analytics, white-label capabilities |
